Раздел 2 · Альтернативы

3 главных экрана без плиток

Если плиточный режим хочется заменить целиком (или сделать как ещё один режим в админке — «Дашборд», «План на день», «Командный центр»).

Альт 1

Dashboard — KPI + график + последние заказы

Никаких плиток-навигаторов. Главный экран = операционная панель: 4 KPI сверху, график выручки, лента последних заказов, быстрые действия. Навигация — через сайдбар (или верхний таб-бар).

★ Раздел 2.5 · Гибриды

Плитки-навигаторы + dashboard-сводка

Сверху — плитки (одним кликом в раздел), снизу — виджеты (видишь операционку без перехода). Лучшее из двух миров: новичок сразу понимает «куда нажать», опытный сразу видит «что делать сейчас».

Гибрид 1

«Лёгкий» · iOS-плитки + 2 виджета

Сверху — компактный ряд из 8 цветных квадратов с бейджами (вариант C, в одну строку). Снизу — два виджета: график выручки и лента заказов. Главное приветствие — небольшое, без лишнего hero.

режим: hybrid · light

Раздел 3 · Резюме

Что я бы выбрал

★ рекомендую

Гибрид 1 — iOS-плитки + 2 виджета

Лёгкий, дышащий, mesh-фон. Плитки в одну строку — навигация на месте, не давит. Снизу — график выручки + лента активности. Минимум доработок (виджеты можно частично переиспользовать со страницы Статистики).

→ переписывает HomeScreen.tsx, добавляет 2 новых виджета
если плотно

Гибрид 2 — плитки + 3 модуля операционки

Самый информативный. Алерт-баннер сверху, узкие плитки-ярлычки 4×2, под ними — «к выдаче / в приёмке / доставка». Подходит если каждый день много операций и приёмщик хочет видеть всё на одном экране.

→ требует хук со списками заказов по статусам
wow / SaaS

Гибрид 3 — Bento с главной Заказы-плиткой

Большая плитка «Заказы 23» с разбивкой 7/13/3 и быстрыми действиями, мини-плитки навигации справа, под ними — график и лента. Самый «современный SaaS»-вид.

→ жёсткий layout, сложнее настраивать через админку
только плитки

Если без виджетов — Вариант A или C

Если решим, что виджеты overkill для главной — лучшие чисто-плиточные варианты: A (цветные с KPI) или C (iOS-style). Минимальные правки HomeScreen.tsx.

Технический момент

Чтобы плитки несли KPI, нужен новый хук useHomeStats(PRODUCT) — он подтянет одной query цифры по разделам (count заказов в работе, count клиентов за месяц, расходники с low_stock). Делается один раз, потом подсказывает любому варианту (A, B, C, D, E, F). Без этого хука даже самая красивая плитка остаётся «иконкой + подписью».
